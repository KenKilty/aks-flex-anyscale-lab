## Cleanup

Run the teardown command to remove all resources created by this lab:

```bash
ANYSCALE_AKS_ENV_FILE=.env ./scripts/anyscale-aks.sh destroy
```

This calls `terraform destroy` against the same variable profile used to deploy,
removes the Flex host, peering, and all managed Azure resources, then verifies
the resource group is empty.

See [Module 7 — Teardown and Replay](/docs/ai-workloads-on-aks/module-07-teardown) for the full ordered teardown runbook.
